## Governance correction

The original P0 documents treated raw-data license resolution and source-by-source human
authorization as a construction gate. The project owner superseded that design on 2026-07-21.
Accepted `ADR-0003` now freezes these rules:

- P0-P7 do not query, infer, compare, approve, or encode raw-data licenses.
- Locally provided research data are selected only through scientific scope and technical checks.
- Source/component records carry only minimal scientific provenance; no permission booleans,
  academic-use flags, reviewer decisions, or authorization requests exist in runtime contracts.
- A future public release of data, benchmark packages, checkpoints, or weights requires a
  separate human publication/release review and is not a Benchmark v3 builder gate.
- Code-copying, dependency-license, attribution, and `NOTICE` obligations remain active.

Historical raw-data permission conclusions in the first P0 audit are therefore non-operative;
the amended `docs/audits/license_matrix.md`, `repo_inventory.json`, and `reuse_matrix.md` are the
current P0 interpretation.
