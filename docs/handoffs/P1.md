## Known risks

- Implemented means only that a sampled layout can be projected safely for audit. It is not a
  claim that the whole source is parsed or licensed.
- GDCLD train/val mixes byte containers and its large test image/mask grids differ. The current
  adapter intentionally covers PNG-signature equal-grid patches only.
- LMHLD four channels have no accepted band identity, and virtual patch rows lack original-scene grouping.
- Sen12 files are 15-step arrays and expose pre/post metadata fields; choosing a time slice without
  source evidence would violate the frozen task scope.
- Landslide4Sense and multimodal metadata were inspected with the user's environment, but those
  readers are not yet declared greenfield dependencies.
- LandslideBench is derived and generated text contains unsupported causal claims. It cannot become truth.
- MMRS aggregate visibility does not license its components; `total.json` remains excluded.
- RSGPT is restricted academic-only and RSIEval remains test-only.
- P1 cannot pass until a real Small benchmark is independently built and validated, which in turn
  requires named human data-license decisions.
