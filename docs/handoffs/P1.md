## Real-data defects found and resolved

1. Local Sen12 stores some missing event dates as literal `None`; these records are now excluded
   at sample level.
2. DIOR-RSVG interleaves forward box-to-phrase and inverse phrase-to-box dialogs and can contain
   multiple pairs per row; P1 selects exactly the forward pairs.
3. All-cloud/all-invalid S2 reference slices are excluded before materialization.
4. Verified duplicate union could absorb the sole hashed validation component; split v2 now moves
   one whole unforced component deterministically to preserve spatial split coverage without
   breaking any grouping constraint.

Each defect has focused regression coverage and the complete suite passed afterward.
