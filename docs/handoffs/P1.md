## Known risks

- The source registry is deliberately fail-closed; public availability or a project root license does not authorize data use.
- The scanner hashes every visible regular file, so a real Full audit may be I/O intensive and must be launched manually.
- A source can appear in inventory while still being unusable for training, evaluation and redistribution.
- JSON Schema cannot express every cross-field spatial invariant as precisely as Pydantic; both validation layers are required.
- P1.2 must not invent invertibility for a transform or silently let unregistered support enter pixel-level evidence.
- Legacy commands remain physically present until later replacement/deletion gates; README labels them as baseline history rather than greenfield runtime.
